# 模板设计规范

*Version 1.1 | 优化版*

本规范定义了基于主题系统的 HTML/CSS 模板的设计原则、布局结构和变量使用标准。适用于读书笔记卡片、金句卡片及其他扩展卡片类型。

---

## 1. 设计原则

- **主题驱动**：所有视觉样式（颜色、字体、间距、圆角、阴影）必须通过 CSS 变量定义，不允许硬编码。
- **固定画布**：卡片画布固定为 **1080×1920 像素**，适配抖音、小红书等竖版平台。
- **响应式滚动**：当内容超过一屏时，卡片内部可滚动，滚动条样式受主题控制。
- **内容优先**：装饰层（渐变光晕、纹理）不影响内容可读性，且应使用 `pointer-events: none`。
- **语义化类名**：使用描述性类名（如 `.book-header`、`.quote-content`），避免歧义。
- **克制优于装饰**：一个卡片只使用一种装饰手法（光晕或纹理，不同时使用多层渐变叠加）。
- **回退可读**：所有 CSS 变量引用必须提供硬编码回退值，确保无主题文件时仍可渲染。

---

## 2. 画布结构

所有卡片模板必须遵循以下 HTML 骨架（**不可随意简化 DOM 层级**）：

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0, user-scalable=yes">
  <title>OpenDesign · 卡片类型 | {{.Title}}</title>
  <style>/* 主题变量 + 布局样式 */</style>
</head>
<body>
  <div class="od-[card-type]-card">      <!-- 根据卡片类型命名，如 od-note-card、od-quote-card -->
    <div class="card-deco"></div>        <!-- 装饰层（可选） -->
    <div class="safe-area">              <!-- 内容安全区，避免滚动时装饰被遮盖 -->
      <div class="card-content">         <!-- 实际内容容器 -->
        <!-- 具体内容区块 -->
      </div>
    </div>
  </div>
</body>
</html>
```

### 2.1 各层职责

| 层级 | 必需 | 定位 | 职责 |
|------|------|------|------|
| `body` | ✅ | static | 固定画布尺寸、外围背景色、整体居中（不承载卡片样式） |
| `od-[type]-card` | ✅ | `position:relative` | 卡片容器，承载背景色、圆角、阴影、滚动 |
| `card-deco` | ❌（可选） | `position:absolute; inset:0` | 装饰层，`pointer-events:none`，z-index 1 |
| `safe-area` | ✅ | `position:relative; z-index:2` | 安全区，内容居中，padding 定义内容与卡片边缘的距离 |
| `card-content` | ✅ | static flex | 内容区块容器，使用 `gap` 定义区块间距 |

### 2.2 禁止的改动

- ❌ 删除 `od-[type]-card` 层级让 `body` 直接当卡片使用
- ❌ 将滚动条设在 `safe-area` 上而非卡片根容器
- ❌ 在 `body` 上设置圆角、阴影、背景色（应委托给卡片容器）

## 3. 主题变量使用规范

模板中每个样式属性必须引用主题变量，并提供合理的回退值（fallback）。回退值应保证在无主题时仍可读。

### 3.1 颜色

| 用途 | 变量 | 回退值示例 |
|------|------|------------|
| 外围背景 | `var(--od-color-bg)` | `#14161f` |
| 卡片背景 | `var(--od-color-surface)` | `#ffffff` |
| 主色（强调） | `var(--od-color-primary)` | `#7c3aed` |
| 辅助色 | `var(--od-color-secondary)` | `#9aa9b9` |
| 标题文字 | `var(--od-color-text-high)` | `#f3f4f6` |
| 正文文字 | `var(--od-color-text-medium)` | `#e5e7eb` |
| 弱化文字 | `var(--od-color-text-low)` | `#a0a0a0` |
| 分割线 | `var(--od-color-divider)` | `#2d2f3e` |
| 强调区块背景 | `var(--od-color-surface-accent)` | `#1f2230` |
| 高亮色（星星等） | `var(--od-color-highlight)` | `#fbbf24` |

### 3.2 字体

- **字族**：`var(--od-font-family, system-ui, ...)`
- **字重**：
  - 常规：`var(--od-font-weight-regular, 400)`
  - 中等：`var(--od-font-weight-medium, 500)`
  - 粗体：`var(--od-font-weight-bold, 700)`
- **字号（使用数字后缀）**：

| 层级 | 变量 | 回退值（px） | 用途 |
|------|------|--------------|------|
| 1（最大） | `var(--od-font-size-1)` | `84` | 超大标题 |
| 2 | `var(--od-font-size-2)` | `40` | 大标题/星星 |
| 3 | `var(--od-font-size-3)` | `36` | 正文 |
| 4 | `var(--od-font-size-4)` | `34` | 副文本/作者 |
| 5 | `var(--od-font-size-5)` | `32` | 注释/标签 |

- **行高**：`var(--od-line-height-body, 1.65)`
- **字间距**：`var(--od-letter-spacing-body, 0.02em)`

### 3.3 间距（gap/margin/padding）

使用语义间距变量，避免魔法数字：

| 变量 | 回退值 | 用途 |
|------|--------|------|
| `var(--od-space-xs)` | `12px` | 极小间距（内边距） |
| `var(--od-space-sm)` | `24px` | 小间距 |
| `var(--od-space-md)` | `40px` | 中间距（常用） |
| `var(--od-space-lg)` | `56px` | 大间距（区块分隔） |
| `var(--od-space-xl)` | `72px` | 超大间距（安全区上下） |

### 3.4 圆角

| 变量 | 回退值 | 用途 |
|------|--------|------|
| `var(--od-radius-sm)` | `16px` | 小圆角 |
| `var(--od-radius-md)` | `36px` | 中圆角（卡片本身） |
| `var(--od-radius-lg)` | `48px` | 大圆角 |
| `var(--od-radius-full)` | `999px` | 胶囊/标签 |

### 3.5 阴影

- **卡片投影**：`var(--od-shadow-card, 0 24px 40px -14px rgba(0,0,0,0.6))`
- 可选内部阴影 `--od-shadow-paper` 按需使用。

## 4. 布局规范

### 4.1 固定尺寸

```css
body {
  width: 1080px;
  height: 1920px;
  margin: 0 auto;
  padding: 100px 150px 140px;
  display: flex;
  justify-content: center;
  align-items: center;
  background: var(--od-color-bg, #14161f);
}
```

> 安全边距确保内容不贴边，同时为装饰层留出空间。上下 padding 可不对称（上 100px 下 140px），左右应对称。

### 4.2 卡片容器

```css
.od-quote-card {
  position: relative;
  width: 100%;
  height: 100%;
  overflow-y: auto;
}
```
>>>卡片的圆角和阴影由具体主题或上层容器决定，不在模板中预设。这确保了卡片可以在不同场景下灵活嵌套（如放入带圆角的父容器、画廊布局等）。
明白，你是要优化规范中 `.od-quote-card` 的 CSS，去掉圆角和阴影的定义。

修改位置：§4.2（布局规范-卡片容器）和 §9（完整示例）。

---

### 4.3 安全区

```css
.safe-area {
  position: relative;
  z-index: 2;
  min-height: 100%;
  display: flex;
  flex-direction: column;
  justify-content: center;
  padding: 40px 60px;
}
```

### 4.4 内容区块间距

内容区块之间使用 `gap: var(--od-space-lg, 56px)`（金句卡片为一屏内仅一句+来源，使用大间距分隔），避免过度使用 `margin-bottom`：

```css
.card-content {
  display: flex;
  flex-direction: column;
  gap: var(--od-space-lg, 56px);
}
```

### 4.5 滚动条

卡片根容器设置 `overflow-y: auto`，并定制滚动条样式：

```css
.od-card::-webkit-scrollbar {
  width: 6px;
  background: transparent;
}
.od-card::-webkit-scrollbar-thumb {
  background: var(--od-color-text-low, rgba(255,255,255,0.2));
  border-radius: var(--od-radius-full);
}
```

## 5. 装饰层规范（可选）

装饰层用于增加视觉层次，但不应干扰内容。

- **位置**：绝对定位，`inset: 0`，覆盖卡片区域。
- **交互**：`pointer-events: none`。
- **克制原则**：只使用一种装饰手法（光晕或纹理），不叠加多层径向渐变。
- **默认样式**：

```css
.card-deco {
  position: absolute;
  inset: 0;
  pointer-events: none;
  background:
    radial-gradient(
      ellipse at 50% 20%,
      color-mix(in srgb, var(--od-color-primary, #7c3aed) 25%, transparent) 0%,
      transparent 70%
    );
  z-index: 1;
  border-radius: inherit;
}
```

- 光晕颜色使用 `var(--od-color-primary)`，透明度建议 25%。
- 如需横线纹理，可单独叠加，但建议作为主题级别的 `background-image` 配置，而非硬编码进模板。
- 装饰层必须设置 `border-radius: inherit` 以匹配父容器的圆角。

## 6. 特殊元素规范

### 6.1 金句模板

- 使用 `.quote-text` 和 `.quote-source`。
- 引号通过 `::before`/`::after` 伪元素添加，颜色取 `var(--od-color-primary)`。
- 出处上方添加分割线：`border-top: 2px solid var(--od-color-divider)`。
- 间距使用 `gap: var(--od-space-lg, 56px)` 分隔金句与来源区块。

```html
<div class="card-content">
  <div class="quote-text">{{.Quote}}</div>
  <div class="quote-source">{{.Source}}</div>
</div>
```

```css
.quote-text {
  font-size: var(--od-font-size-1, 84px);
  font-weight: var(--od-font-weight-bold, 700);
  line-height: var(--od-line-height-body, 1.65);
  color: var(--od-color-text-high, #f3f4f6);
  letter-spacing: var(--od-letter-spacing-body, 0.02em);
  white-space: pre-wrap;
  word-break: break-word;
}
.quote-text::before {
  content: "“";
  color: var(--od-color-primary, #7c3aed);
  font-size: 1.2em;
  opacity: 0.6;
  margin-right: 0.1em;
}
.quote-text::after {
  content: "”";
  color: var(--od-color-primary, #7c3aed);
  font-size: 1.2em;
  opacity: 0.6;
  margin-left: 0.1em;
}
.quote-source {
  font-size: var(--od-font-size-4, 34px);
  font-weight: var(--od-font-weight-medium, 500);
  color: var(--od-color-primary, #7c3aed);
  border-top: 2px solid var(--od-color-divider, #2d2f3e);
  padding-top: var(--od-space-md, 40px);
  display: inline-block;
  align-self: center;
}
```

### 6.2 读书笔记模板

- 书籍标题使用 `.book-title`，字号 `var(--od-font-size-1)`。
- 作者、评分使用 `.author-rating` 容器，内部元素分别应用对应字号变量。
- 笔记正文使用 `.reflection`，内部段落 `<p>` 自动生成，高亮 `<strong>` 颜色为 `var(--od-color-primary)`。
- 区块间距：标题与作者评分为 `var(--od-space-md)`，评分与正文为 `var(--od-space-lg)`。

## 7. 扩展性

- **新增卡片类型**：只需复制现有模板，修改根容器类名和内容区域，样式复用主题变量。
- **主题覆盖**：若某个主题需要微调特定卡片类型的样式，可以通过在主题中增加 `--od-<card-type>-*` 变量实现，但需在模板中预留。
- **媒体查询**：针对极小屏幕（如手机截图），可添加 `@media` 规则微调间距和字号，但应尽量保持主题变量不变。

---

## 8. 常见违规模式（禁止清单）

| 违规 | 后果 | 正确做法 |
|------|------|----------|
| 删除 `od-[type]-card` 容器，body 直接做卡片 | 装饰层无法正确定位，滚动条丢失 | 保留完整骨架 |
| 引号改用内联 `<span>` 而非 `::before/after` | 主题无法通过 CSS 控制引号颜色/大小 | 使用 `::before/::after` 伪元素 |
| 分割线改用独立 `<div>` 而非 `border-top` | 增加无用 DOM 节点，主题变量不适用 | 使用 `border-top: 2px solid var(--od-color-divider)` |
| 多层装饰叠加（双径向渐变 + 纹理 + 噪点） | 干扰内容可读性，视觉杂乱 | 只使用一种装饰手法 |
| 使用 `gap: 0` + 手动 margin 控制间距 | 累积边距导致排版偏移 | 使用语义化 `gap` 变量 |
| 缺少 CSS 变量 fallback | 无主题文件时页面白屏 | 每个 `var()` 都提供硬编码回退值 |

## 9. 完整示例：金句卡片模板（v1.1 合规）

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0, user-scalable=yes">
    <title>OpenDesign · 金句卡片 | {{.Quote}}</title>
    <style>
        * { margin: 0; padding: 0; box-sizing: border-box; }

        body {
            width: 1080px;
            height: 1920px;
            margin: 0 auto;
            padding: 100px 150px 140px;
            background: var(--od-color-bg, #14161f);
            font-family: var(--od-font-family, system-ui, -apple-system, 'Segoe UI', 'PingFang SC', 'Microsoft YaHei', 'Helvetica Neue', sans-serif);
            display: flex;
            justify-content: center;
            align-items: center;
        }

        .od-quote-card {
            position: relative;
            width: 100%;
            height: 100%;
            overflow-y: auto;
            background: var(--od-color-surface, #ffffff);
            border-radius: var(--od-radius-md, 36px);
            box-shadow: var(--od-shadow-card, 0 24px 40px -14px rgba(0,0,0,0.6));
        }

        .od-quote-card::-webkit-scrollbar {
            width: 6px;
            background: transparent;
        }
        .od-quote-card::-webkit-scrollbar-thumb {
            background: var(--od-color-text-low, rgba(255,255,255,0.2));
            border-radius: var(--od-radius-full, 999px);
        }

        .card-deco {
            position: absolute;
            inset: 0;
            pointer-events: none;
            background: radial-gradient(
                ellipse at 50% 20%,
                color-mix(in srgb, var(--od-color-primary, #7c3aed) 25%, transparent) 0%,
                transparent 70%
            );
            z-index: 1;
            border-radius: inherit;
        }

        .safe-area {
            position: relative;
            z-index: 2;
            min-height: 100%;
            display: flex;
            flex-direction: column;
            justify-content: center;
            padding: 40px 60px;
        }

        .card-content {
            display: flex;
            flex-direction: column;
            gap: var(--od-space-lg, 56px);
            text-align: center;
        }

        .quote-text {
            font-size: var(--od-font-size-1, 84px);
            font-weight: var(--od-font-weight-bold, 700);
            line-height: var(--od-line-height-body, 1.65);
            color: var(--od-color-text-high, #f3f4f6);
            letter-spacing: var(--od-letter-spacing-body, 0.02em);

            word-break: break-word;
        }

        .quote-text::before {
            content: "“";
            color: var(--od-color-primary, #7c3aed);
            font-size: 1.2em;
            opacity: 0.6;
            margin-right: 0.1em;
        }
        .quote-text::after {
            content: "”";
            color: var(--od-color-primary, #7c3aed);
            font-size: 1.2em;
            opacity: 0.6;
            margin-left: 0.1em;
        }

        .quote-source {
            font-size: var(--od-font-size-4, 34px);
            font-weight: var(--od-font-weight-medium, 500);
            color: var(--od-color-primary, #7c3aed);
            border-top: 2px solid var(--od-color-divider, #2d2f3e);
            padding-top: var(--od-space-md, 40px);
            display: inline-block;
            align-self: center;
        }
    </style>
</head>
<body>
    <div class="od-quote-card">
        <div class="card-deco"></div>
        <div class="safe-area">
            <div class="card-content">
                <div class="quote-text">{{.Quote}}</div>
                <div class="quote-source">{{.Source}}</div>
            </div>
        </div>
    </div>
</body>

```
