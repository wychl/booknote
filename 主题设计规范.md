# 主题设计规范

*Version 2.0 | 基于通用语义令牌*

本规范定义了主题系统的设计原则、变量体系、文件结构及创建新主题的标准流程。适用于读书笔记卡片、金句卡片及其他任何需要统一视觉风格的场景。

---

## 1. 设计原则

- **语义化命名**：变量名描述用途而非具体场景（如 `--od-color-text-high` 而非 `--od-color-book-title`）。
- **层级清晰**：文字大小使用数字后缀 1~5（从大到小），间距使用 `xs` ~ `xl` 梯度。
- **主题自包含**：一个主题 JSON 文件包含所有视觉变量，不依赖外部默认值。
- **色彩和谐**：`--od-color-secondary` 应基于 `--od-color-primary` 同色系演变，避免高反差撞色。
- **易扩展**：新增主题只需复制现有 JSON 并修改颜色值，无需改动代码。

---

## 2. 变量体系

### 2.1 颜色变量（17个，必填）

| 变量 | 用途 | 示例值（深色主题） |
|------|------|-------------------|
| `--od-color-bg` | 卡片外围背景 | `#0a0a0f` |
| `--od-color-surface` | 卡片主体背景 | `#12121a` |
| `--od-color-primary` | 主品牌色 | `#c084fc` |
| `--od-color-primary-dark` | 主色暗色变体 | `#a855f7` |
| `--od-color-secondary` | 辅助色（应与主色协调） | `#d8b4fe` |
| `--od-color-hover` | 可交互元素悬停色 | `#c084fc` |
| `--od-color-link` | 链接颜色 | `#d8b4fe` |
| `--od-color-divider` | 分割线/边框色 | `#2d2d3a` |
| `--od-color-text-high` | 高强调文字（标题） | `#ffffff` |
| `--od-color-text-medium` | 中强调文字（正文） | `#e2e2ea` |
| `--od-color-text-low` | 低强调文字（辅助） | `#a1a1aa` |
| `--od-color-surface-accent` | 强调区块背景 | `color-mix(in srgb, var(--od-color-primary) 15%, var(--od-color-surface))` |
| `--od-color-highlight` | 高亮色（评分、标记等） | `#fbbf24` |
| `--od-color-success` | 成功状态 | `#9ece6a` |
| `--od-color-error` | 错误状态 | `#f7768e` |
| `--od-color-warning` | 警告状态 | `#e0af68` |

> **注意**：`--od-color-surface-accent` 推荐使用动态混合色，始终保持与主色的微弱关联。

### 2.2 字体尺寸变量（5个，必填）

| 变量 | 推荐值 | 用途 |
|------|--------|------|
| `--od-font-size-1` | `84px` | 超大标题（一级） |
| `--od-font-size-2` | `40px` | 大标题 / 星星 |
| `--od-font-size-3` | `36px` | 正文 |
| `--od-font-size-4` | `34px` | 副文本 / 作者 |
| `--od-font-size-5` | `32px` | 注释 / 标签 |

所有主题必须定义这 5 个变量，且推荐使用上述值以保持各卡片尺寸一致。如需特殊缩放（如极小屏幕），可通过媒体查询覆盖。

### 2.3 间距变量（5个，必填）

| 变量 | 推荐值 | 用途 |
|------|--------|------|
| `--od-space-xs` | `12px` | 极小间距 |
| `--od-space-sm` | `24px` | 小间距 |
| `--od-space-md` | `40px` | 中间距 |
| `--od-space-lg` | `56px` | 大间距 |
| `--od-space-xl` | `72px` | 超大间距 |

### 2.4 圆角变量（4个，必填）

| 变量 | 推荐值 | 用途 |
|------|--------|------|
| `--od-radius-sm` | `16px` | 小圆角 |
| `--od-radius-md` | `36px` | 中圆角 |
| `--od-radius-lg` | `48px` | 大圆角 |
| `--od-radius-full` | `999px` | 完全圆角（胶囊） |

### 2.5 字体相关（必填）

| 变量 | 推荐值 |
|------|--------|
| `--od-font-family` | `system-ui, -apple-system, 'Segoe UI', 'PingFang SC', 'Microsoft YaHei', 'Helvetica Neue', sans-serif` |
| `--od-font-weight-regular` | `400` |
| `--od-font-weight-medium` | `500` |
| `--od-font-weight-bold` | `700` |
| `--od-line-height-body` | `1.65` |
| `--od-letter-spacing-body` | `0.02em` |

### 2.6 阴影变量（可选，但强烈建议）

| 变量 | 示例值 |
|------|--------|
| `--od-shadow-card` | `0 24px 40px -14px rgba(0, 0, 0, 0.75)` |

根据明暗主题调整透明度：亮色主题 `0.08~0.2`，深色主题 `0.6~0.85`。

## 3. 主题文件结构

每个主题是一个独立的 JSON 文件，放置于 `internal/theme/themes/` 目录，文件名为 `<id>.json`。

**模板**：

```json
{
  "id": "your-theme-id",
  "name": "主题显示名称",
  "description": "简短描述（适用场景、风格特点）",
  "variables": {
    "--od-color-bg": "...",
    "--od-color-surface": "...",
    // ... 所有上述必填变量
  }
}
```

**命名规范**：

- `id`：小写字母、数字、连字符，如 `black-gold`、`cyberpunk`。
- `name`：中文，2~6 个字，如 `黑金尊享`。
- `description`：一句话，不超过 50 字。

---

## 4. 创建新主题的步骤

1. **复制模板**：从现有主题（如 `black-gold.json`）复制一份。
2. **修改元数据**：更改 `id`、`name`、`description`。
3. **调整颜色变量**：
   - 保持 `--od-color-surface` 与 `--od-color-bg` 的对比度适宜。
   - `--od-color-primary` 选取主题主色调。
   - `--od-color-secondary` 选择主色的同色系较亮或较暗版本（避免高反差）。
   - 其他功能色（success/error/warning）可沿用默认推荐值或根据主色微调。
4. **保留其他变量**：字体尺寸、间距、圆角、字体族等**不修改**，以确保各主题卡片尺寸统一。
5. **设置阴影**：根据主题明暗选择合适的 `--od-shadow-card`。
6. **测试**：
   - 运行 `booknote themes` 确认主题被加载。
   - 生成一张卡片预览效果：`booknote quote "测试" --source "测试" --theme your-id --image`。

---

## 5. 最佳实践

- **深色主题**：
  - `--od-color-bg` 接近 `#000000`，`--od-color-surface` 稍亮（如 `#0a0a0a`）。
  - `--od-color-text-high` 使用 `#ffffff`，`--od-color-text-medium` 用 `#e0e0e0`。
  - 阴影透明度 `0.8` 以上。

- **浅色主题**：
  - `--od-color-bg` 用浅灰（`#f0f4f8`），`--od-color-surface` 用白色。
  - 文字颜色深色，如 `#0f172a`。
  - 阴影透明度 `0.1~0.2`。

- **强调色使用**：`--od-color-primary` 用于重要元素（标题、链接、按钮），`--od-color-secondary` 用于次要强调（副文本、图标）。

- **动态背景**：`--od-color-surface-accent` 始终使用 `color-mix(in srgb, var(--od-color-primary) 15%, var(--od-color-surface))`，保证主题一致性。
